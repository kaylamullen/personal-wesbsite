Multi-Task Allocation for Mobile Crowd Sensing Platforms

Final project for CS236: Spaital-Temporal Design — Wellesley College

This project was completed as the final project for CS236 at Wellesley College. The official project description is not publicly available, however, I'm happy to provide additional details and discuss the work one-on-one upon request.

Code and detailed materials available upon request. Contact me

Overview

Developed an algorithm for allocation of spatial-temporal tasks to participants in different locations to optimize total possible task reward in the system. Implemented a simulator to test algorithm performance on real data. Trained Prophet models on time-series data for each location to predict future task rewards.
